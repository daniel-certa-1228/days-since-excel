## How to Use

### Deployment
1. Upload `days-since.html` to your web hosting service, GitHub Pages, Netlify, or wherever
2. Send the link to everyone except Dan
3. Wait

### When Dan Inevitably Forgets Again

1. Open `days-since.html` in any text editor
2. Find this line:
   ```javascript
   const LAST_INCIDENT_DATE = '2025-01-15';
   ```
3. Update the date to today's date (format: `YYYY-MM-DD`)
4. Commit and push
5. Send the link to Dan
6. Repeat in 2-3 weeks

## Technical Details

For those who care (looking at you, Dan, who should be updating that Export instead of reading this):

- Pure vanilla JavaScript
- CSS3 gradients and flexbox
- Calculates days since midnight of the incident date
- Counter updates automatically every hour
- Works in all modern browsers (and probably IE11, but we don't talk about that)
